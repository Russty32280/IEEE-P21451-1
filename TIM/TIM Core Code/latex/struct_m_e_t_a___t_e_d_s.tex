\subsection{Detailed Description}
Contains all basic Information for the T\+I\+M; calls upon the \hyperlink{_t_i_m1_8c_ab0bf2da84344cc59e76fdf07beb14e73}{Meta\+T\+E\+D\+S\+Init()} function in \hyperlink{_t_i_m1_8h}{T\+I\+M1.\+h}. 

All T\+I\+Ms on a network need a T\+E\+D\+S to tell the N\+C\+A\+P what exactly it is capable of. The first type of T\+E\+D\+S the N\+C\+A\+P will attempt to read is the Meta\+T\+E\+D\+S which contains information about how many sensor channels there are, worst case sampling times, as well as other communication and initialization information. The fields and their applicable data types can be found in the I\+E\+E\+E P21451-\/2 documentation.
